# Mini App Examples

This repository contains example mini apps that can be used with the Movement Mini App SDK.

## Mini App SDK

The Movement Mini App SDK is a library that allows you to integrate mini apps into your app.

## Mini App Examples

The mini app examples are a set of mini apps that can be used with the Movement Mini App SDK.

### Mini App Send Tokens

The Mini App Send Tokens is a mini app that allows users to send tokens to other users.

### Mini App Claim

The Mini App Claim is a mini app that allows users to claim tokens from a smart contract.
